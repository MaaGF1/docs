# 自动打灰彩蛋关

## 一、核心功能

1. 自动打`彩蛋关卡`获取灰域积分；
2. 全人形自动拆解/二星人形自动拆解。

## 二、使用方式

- 主菜单开启
   1. 确认有`灰域探查`按钮；
   2. 点击脚本开始运行。
- 灰域界面运行
   1. 手动进入灰域，推荐`硅状带`；
   2. 点击脚本开始运行。

## 三、注意

1. 炸山功能需手动操作，自动炸山可以选择`DLZ版自动灰域`；
2. 已解锁硅状带区域；
3. 有足够的行动点数；
4. 代理作战功能已开启

## 四、收益

1. 四项(或等价探查点)消耗：`-800/h`
2. 灰域积分获得：`2000/h`
