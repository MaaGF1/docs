# 零元购使用手册

## 一、普通版

### 1.1 莫比乌斯带上的逃亡

#### 如何运行

- 自动运行
    1. 直接从主界面开始运行，脚本自动完成缩放；
- 手动运行(自动运行**识别失败**)
    1. 进入卷积核**普通难度**的打捞关卡`莫比乌斯带上的逃亡`；
    2. 将`地图缩放到最小`且`向右上拉不动`为止，部署重装后开始运行。

#### 注意事项

进入活动界面后若**没有识别到**关卡会有个缩放动作，可手动缩放到最小即鼠标滑轮向下，再向下拖拽到底，确保关卡大概位于中间范围即可。另外在虚拟机上貌似只能手动操作前述动作。

#### 粗略收益

1. 人力：812/H
2. 弹药：847/H
3. 口粮：613/H
4. 零件：195/H

### 1.2 黑暗中的对话

#### 如何运行

- 自动运行
    1. 直接从主界面开始运行，脚本自动完成缩放；
- 手动运行(自动运行**识别失败**)
    1. 进入卷积核**普通难度**的关卡`黑暗中的对话`；
    2. 将`地图缩放到最小`且`向右上拉不动`为止，部署重装后开始运行。

#### 注意事项

缩放要求同`莫比乌斯带上的逃亡`。

#### 粗略收益

待测。

## 二、AD零元购

AD零元购要求使用`-Agent`后缀版本的`MaaGF1`。

该版本增加了定时查收后勤的功能，要求只能从主界面开始运行。
