## 1.2 注意

1. 铁血梯队不应配备“鬼步”（鬼影姿态）芯片；不应该配备“多带粮弹”（充足燃料）芯片。
2. 如果在“补给阶段”（见下方状态机）卡住，请关闭脚本手动执行补给，并结束本次循环；
3. 每次补给一个铁血梯队之后，会有3秒左右的等待，属于正常现象。
