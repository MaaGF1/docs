# 13-4双打手拖尸使用手册

***（以下内容均以默认设置为准，13-4的其他可选项说明在该文档底）***

---

## 🛠 准备工作清单

### ⚙️ 游戏设置

- 进入 **设置 → 基础**：
  - **分辨率**：1280×720
- 进入 **设置 → 作战**：
  - **记录上次完成关卡时镜头缩放**：开启
  - **自动补给**：关闭

### 👥 梯队配置

1. **第一梯队**：
   - 打手在梯队在梯队 👉 ***非队长位*** 👈
   - 任意一个妖精（确保打手能正常炸翻），不得为空

2. **第二梯队**：
   - 仅包含打手,**不能包含妖精！****不能包含妖精！****不能包含妖精！**

> **关键要求**：
>
> - ✅ 确保其中一个打手满弹药，另一个打手空弹药
>
> - ✅ 将两个打手添加为收藏人形（确保不翻页即可找到）

### 🔒 人形拆解警告

```
⚠️ 脚本会清除所有未上锁人形！
请确认所有需保留人形均已上锁且无可拆的五星人形！
```

### 🗺 地图初始设置

1. 进入13-4关卡
2. 将地图**居中以后再缩放到最小**，运行一次初始化或手动拉到最右上方：

**地图布局必须和这张图一致！若无法调整至这个缩放，请先把地图居中、缩放到最小后再拖动地图！**

梯队部署（可以交给脚本自动部署）：

- **指挥部（HQ）**：放置第一梯队
- **左机场**：放置第二梯队

### ✅ 最终验证

**强烈建议首次运行时进行该操作，以避免自动初始化在不同设备上的不同表现带来的误差**

1. 按照示意图手动执行一次拖尸操作，确认：
   - 打手能正常击败敌人，梯队不会被击败

2. 点击**再次作战**，确认：
   - 地图缩放比例保持不变
   - 无需拖动地图即可开始

## 👓 Q&A 常见问题解答

### ❓ Q：进入13-4点击开始任务没反应？

```
⚠️ 解决方案：地图缩放与位置问题
```

#### ✅ 正确操作步骤

1. **居中地图**：
   - 确保地图视图大致居中
   - 非居中状态可能导致缩放异常

2. **确认重型机场位置**：

   ```位置要求
   重型机场应位于左下角两个UI元素的夹缝中间
   ```

3. **缩放至最小**：
   - 使用鼠标滚轮或缩放按钮缩放到最小级别

---

### ❓ Q：部署梯队后卡住？

```
🔧 解决方案：弹药检查版本问题
```

#### ✅ 操作步骤

```设置路径
选项 → 选择"新版弹药检查"
```

> *提示：确保一个打手满弹药，另一个打手空弹药*

## 13-4 选项说明

| 选项名称 | 选项说明 |
| :--- | :--- |
| **134\_弹药检查版本选择** | **控制弹药识别逻辑版本**<br>- 启用新版弹药识别（默认）：地图界面检测弹粮圈是否为空<br>- 启用旧版弹药识别：点进梯队详情检测弹药条是否为空 |
| **134\_是否开启结算加速** | **控制战斗结算是否加速**<br>- 启用结算加速（默认）：识别到战斗胜利界面下方的EXP时进入加速结算流程，快速连点缩短结算时间<br>- 关闭结算加速：点击执行计划后固定等待65秒，适用于结算加速频繁会跳出“是否终止计划”弹窗的用户 |
| **134\_是否开启无限循环** | **控制任务重复次数**<br>- 开启无限循环：任务完成后自动重启“13-4循环”（不限制次数）<br>- 指定次数（满仓停止）：仅执行设定次数，满仓后停止 |
| **134\_路线选择** | **控制作战路线（不同路线需要按情况打开自动补给）**<br>1. 五战路线（拖尸默认方案，需部署二队）：自动补给：**关闭**<br>2. 四战路线（不部署二队，需开启自动补给）：自动补给：**开启**<br>3. 三战路线（不部署二队，需开启自动补给）：自动补给：**开启** |
