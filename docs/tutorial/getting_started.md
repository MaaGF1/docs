# MaaGFL 快速开始指南

&emsp;&emsp;本指南将详细介绍如何下载、安装并启动 MaaGFL 辅助工具，以及进行必要的配置。

## 一、下载 MaaGFL

&emsp;&emsp;您可以在 [MaaGFL GitHub Releases 页面](https://github.com/MaaGF1/MaaGF1/releases) 找到所有历史发行版本。其中，带有 `latest` 标签的版本是最新稳定版，而 `Pre-release` 则表示预发行版本。

&emsp;&emsp;建议您访问 [最新稳定版页面](https://github.com/MaaGF1/MaaGF1/releases/latest)。在该页面的 `Assets` 一栏中，找到名为 `MaaGF1-GUI-vA.B.C-ARCH-.zip` 的压缩包并下载。
*   `vA.B.C` 代表软件的版本号，例如 `v1.2.3`。
*   `ARCH` 指代您的系统架构，目前支持 `aarch64` 或 `x86-64`。请根据您的操作系统选择正确的版本。

## 二、运行 MaaGFL

&emsp;&emsp;下载 `.zip` 压缩包后，请将其解压到您选择的目录。然后，运行解压后的文件夹中的 `MFAAvalonia.exe` 程序。运行 MaaGFL 需要以下依赖环境：

*   `.Net Runtime` 版本高于 8.0（适用于 MaaGFL 1.7 版本）
*   `.Net Runtime` 版本高于 10.0（适用于 MaaGFL 1.8 及后续版本）
*   `VC++` 版本高于 14.4

> **提示：** 从 MaaGFL 1.8 版本开始，解压后的根目录中会包含一个名为 `DependencySetup_依赖库安装_win.bat` 的脚本，运行它可以帮助您一键安装所需依赖。

&emsp;&emsp;首次运行 `MFAAvalonia.exe` 后，您可能会看到一系列提示信息，例如“开始查找窗口”、“已选择窗口”、“软件已是最新或有新版本”以及“该资源不支持Mirror酱”等。

&emsp;&emsp;MaaGFL 界面包含多个重要子窗口，理解它们的功能对于顺畅使用至关重要：

*   **连接**：在此处确保“当前控制器”已正确选择到您的《少女前线》游戏窗口。**注意**：在 MaaGFL 1.8 及更高版本中，您需要手动指定游戏窗口。
*   **资源类型**：选择您正在玩的游戏服务器，目前支持国服和美服。
*   **任务列表**：浏览并选择 MaaGFL 提供的各种自动化脚本。
*   **任务设置**：针对所选任务进行详细配置，例如指定打捞特定人形等。
*   **任务说明**：提供当前选中任务脚本的简要介绍。
*   **日志**：用于实时查看 `MaaFw` 或 `Agent` 的运行日志信息。

## 三、游戏内设置

&emsp;&emsp;为了确保 MaaGFL 能够正常工作，请在《少女前线》游戏中进行以下配置：

1.  将游戏设置为**窗口模式**，并将分辨率调整为 `1280x720`。
2.  在游戏设置中，勾选“记录上次完成关卡时镜头缩放”选项。

按照以上步骤，您就可以开始使用 MaaGFL 辅助您的游戏体验了！
